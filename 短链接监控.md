# 短链接监控

跳转的同时提供监控信息：访问次数----每跳转一次就记录一次     pv
                        访问人数----根据用户  uv  
                        IP数--------统计同一IP下访问次数  uip
                        访问地区----  中国地图 调用高德api
                        24小时分布--用户访问的时间 
                        高频IP------记录访问次数高的
                        一周访问----具体周几
                        操作系统----windows linux macOS
                        访问浏览器--firefox chrome edge
                        访客类型----分为新老访客
                        访问网络----wifi 移动数据
                        访问设备----电脑 移动设备
